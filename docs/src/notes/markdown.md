# Markdown语法

## 标题，可以生成锚点目录

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```


## 段落与格式
### 段落
Markdown 段落没有特殊的格式，直接编写文字就好，**段落的换行是使用两个以上空格加上回车。**

### 字体
|格式| 例子 | 代码|
|--|--|--|
|斜体|*斜体*|\*斜体文本\*|
|粗体|**粗体**|\*\*粗体文本\*\*|
|粗斜体|***粗斜体***|\*\*\*粗斜体文本\*\*\*|
|删除线|~~删除线文本~~|\~\~删除线文本\~\~|
|下划线|<u>下划线文本</u>|\<u>下划线文本<\/u>|

### 分隔线

```
***
- - -
```
***
- - -
